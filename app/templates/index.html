This website is the part of a Project towards credits for IS601 - Web Systems Development. This is a really nice course to take. We really appreciate Professor Williams Keith to give us the projects that we are been given so far in this course. In the first month itself, We have learnt almost a dozen new concepts of this fast growing technology in Web development.

This Website is about History of the Internet and Web calculator developed using Python, Flask, NGINX. The content of the website summarizes The True Story of the Internet ,Early Innovation Of The Internet ,Early Home Computer, Pylint Introduction, AAA Testing With Python, Principles Of Object-Oriented Programming and Design Patterns documentaries in details. There is menu which redirects to Web Calculator to perform basic operations like Addition, Subtraction, Multiplication and Division. Result page displays history of all the operations performed.
